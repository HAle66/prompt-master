# AI Prompt Hub 🚀

The ultimate collection of AI prompts for ChatGPT, Claude, and other AI tools.

## Features

✨ **100+ Curated Prompts** - Ready-to-use prompts for every use case
🎯 **Multiple Categories** - Writing, Marketing, Development, Business, and more
🔍 **Smart Search** - Find the perfect prompt instantly
📋 **One-Click Copy** - Copy prompts to clipboard instantly
🔓 **Free + Premium** - Start free, unlock advanced features

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Modern styling
- **Vercel** - Deployment

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
ai-prompts/
```
